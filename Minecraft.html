World options and Gamemodes

Currently in Minecraft Java Edition there are 5 gamemodes. Survival, Hardcore, Creative, adventure and spectator.

Survival mode is one of the allowed starting modes. Survival mode allows you to explore with freedom. You can change your game diffculty and can use cheats if desired. Hardcore is one of the allowed gamemodes but is more challenging. Hardcore is locked at the ahrdest diffculty and cant use cheats. This is mostly used in 100 day hardcore videos. Creative mode has more freedom and can build anything without problems Spectator and adventure mode are only able to be used in a world with cheats enabled. heres how to enable them with cheats /gamemode survival (player name) /gamemode creative (Player name) /gamemode spectator (player name) /gamemode adventure (player name)
